HTML: Hypertext Markup Language (Lenguaje de Marcado de Hipertexto)

HTML, es el lenguaje estándar utilizado para crear el "esqueleto" o la estructura de las páginas web.

HTML funciona mediante una estructura de árbol, donde el nodo raíz es la etiqueta "html", y dentro de ella se anidan otros nodos o hijos" que forman la jerarquía del documento

Componentes de un Documento HTML

Un documento HTML5 estándar se compone de secciones fundamentales: !DOCTYPE html, head, body.

Es importante diferenciar que el CSS es la herramienta que describe la presentación y el estilo visual de ese documento. Un mismo archivo HTML puede verse de formas completamente distintas dependiendo del código CSS que se le aplique

Etiquetas de Estructura Básica

Para que el contenido sea comprensible tanto para desarrolladores como para motores de búsqueda, HTML utiliza etiquetas semánticas que agrupan la información lógicamente.

html: Es la etiqueta raíz que envuelve todo el documento.
head: Contiene metadatos y descripciones que no son visibles directamente para el usuario, pero sirven para incluir otros códigos.
body: Contiene todo el contenido visible de la web (texto, imágenes, etc.).
title: Define el título que aparece en la pestaña del explorador y es fundamental para los motores de búsqueda.

Etiquetas de Texto y Formato

Las etiquetas de texto y formato en HTML se utilizan para organizar el contenido de manera jerárquica y semántica, permitiendo que los motores de búsqueda y los navegadores comprendan la estructura de la información.

* Etiquetas de Selección y Comentarios.

* Etiquetas de Párrafo y Saltos.

* Listas y Tablas.
